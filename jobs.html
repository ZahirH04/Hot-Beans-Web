Job Specifications

Web Developer

A web developer would mainly be creating/programming the actual websites once the designs have been completed and it is ready to be executed. This role requires a person with some experience or someone who has done a web development course.

​

Web Designer

A web designer would design the websites, how they look and make a rough version of the website using shapes text and pictures to give a good idea as to how the site would look whilst meeting the needs of the client. This is all done before the execution phase where the web developers would create this site and make it look like the design.​

Web Analyst

A web analyst would review the design to make sure the design is right and meets the requirements of the client then approves it and checks the site once complete to see if it has been done correctly according to the design so that the site is of a good standard and pleases the client. For this role, you need to have a degree in website development and have at least 3 years in website development.

Web Identifier

A web identifier would be the person to identify the needs and wants of the client of what the site should include and the target audience of the client so that some idea of how it should look is given. This is the first phase and is very important for the project to be done correctly. Once done the details would be sent to a web designer which then design a site based off the information provided by the web identifier​. No qualifications are required.
